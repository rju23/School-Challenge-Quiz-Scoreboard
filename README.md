## **üìÑ Description**

This is a **real-time quiz scoreboard platform** for secondary school competitions. It includes:

- A **main scoreboard** display for students.
- A **control panel** for teachers/coaches to manage scores, rounds, and timers.
- Keyboard shortcuts and visual feedback for fast scoring.
- Compatibility with any modern web browser.

---

## **üéÆ Features**

- Three types of rounds:
  - **ALTERNATE** ‚Äì 1 point per correct answer.
  - **SPEED** ‚Äì 1 point per correct answer, timed per minute.
  - **BUZZER** ‚Äì 2 points for correct, -2 for wrong.
- **Keyboard shortcuts**:
  - Team A Correct: `1`
  - Team A Wrong: `2`
  - Team B Correct: `8`
  - Team B Wrong: `9`
- **Timers** for rounds and questions:
  - Alternate: 4mins
  - Speed: 1min
  - Buzzer: 4mins
  - Alternate: 10s per question
  - Buzzer: 5s per question
- **Team score tracking** with correct/wrong marks visible on the control panel.
- **Query deductions** for alternate and speed rounds.

---

## **üñ• How to Use**

1. Open the **control panel** (`controlpanel.html`) to manage the game.
2. Set **team names** in the control panel settings and click "Update Names".
3. Select the **round/section** currently being played.
4. Use **Correct/Wrong buttons** or the assigned **keyboard shortcuts** to update scores.
5. Use the **question timer** for timed questions. Start/restart with `R/r`, pause with `P/p`.
6. Scores will automatically update on the **main scoreboard** (`mainscoreboard.html`) for the teams to see.
